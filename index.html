Sobre Mim

Olá! Meu nome é Morgana Kuhnen, tenho 30 anos e sou Designer e entusiasta do mundo digital. Minha jornada me levou a mergulhar no universo do desenvolvimento Front-end, onde tenho a oportunidade de unir minha paixão pelo design e pela tecnologia.Estou aprendendo HTML, CSS, JavaScript e Frameworks, como o Bootstrap e também em breve React.js. Desde que comecei a minha jornada no desenvolvimento web, descobri uma fascinação pelo poder de transformar ideias criativas em realidade. O que me encanta é a capacidade de moldar interfaces visualmente atraentes e funcionais. A cada dia, busco o aprimoramento da minha capacidade, através de cursos, como este do +praTI, livros e onde mais possa aprender e treinar meus conhecimentos. Cada linha de código que escrevo é um passo em direção a aprimorar minha capacidade de criar experiências digitais. Aprender é uma jornada emocionante e estou comprometida em me manter atualizada. Quando não estou imersa no mundo do código, adoro explorar inspirações visuais, experimentar novas abordagens de design e estudar as melhores práticas de UX/UI. Estou animada para continuar minha jornada como desenvolvedora front-end, explorando novos desafios e criando interfaces que tornem a web um lugar mais bonito e funcional.

CSS 3

Cascading Style Sheets é um mecanismo para adicionar estilos a uma página web, aplicado diretamente nas tags HTML ou ficar contido dentro das tags style.

JavaScript

JavaScript é uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma.

Bootstrap 5

Bootstrap é um framework web com código-fonte aberto para desenvolvimento de componentes de interface e front-end para sites e aplicações web, usando HTML, CSS e JavaScript, baseado em modelos de design para a tipografia, melhorando a experiência do usuário em um site amigável e responsivo.
